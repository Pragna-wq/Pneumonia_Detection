# 🩺 Pneumonia Detection System

## 📌 Overview

The Pneumonia Detection System is a deep learning-based application designed to identify pneumonia from chest X-ray images. The model leverages Convolutional Neural Networks (CNNs) to perform binary image classification (Pneumonia vs Normal) with high accuracy.

---

## 🎯 Objectives

* Automate pneumonia detection using medical imaging
* Assist in early diagnosis and reduce manual effort
* Provide a simple and user-friendly interface for predictions

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries/Frameworks:** TensorFlow, Keras, OpenCV, NumPy
* **Model Type:** Convolutional Neural Network (CNN)

---

## 🚀 Features

* Upload chest X-ray images for analysis
* Predict whether the patient has pneumonia or not
* Fast and efficient image processing
* Easy-to-use interface

---

## ⚙️ Installation & Setup

```bash
git clone https://github.com/Pragna-wq/Pneumonia_Detection.git
cd Pneumonia_Detection
pip install -r requirements.txt
python app.py
```

---

## 📊 Model Details

* Utilizes CNN architecture for image classification
* Trained on labeled chest X-ray datasets
* Optimized for binary classification (Pneumonia / Normal)
