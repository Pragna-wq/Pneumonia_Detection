# Pneumonia Detection System

## 📌 Description
This project uses Deep Learning (CNN) to detect pneumonia from chest X-ray images.

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV

## 🚀 Features
- Image upload
- Pneumonia prediction
- Accuracy display

## ▶️ How to Run
pip install -r requirements.txt
python app.py
